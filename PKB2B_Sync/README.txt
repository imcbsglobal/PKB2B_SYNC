PKB2B Sync Tool
================

Steps to use:
1. Open  config.json  in Notepad.
2. Set  "database"  to your SQL Anywhere database engine name.
3. Set  "base_url"  to the server API address provided to you.
4. Double-click  PKB2B_Sync.exe  to start syncing.

Note: The SQL Anywhere database must be running before you start the sync.
